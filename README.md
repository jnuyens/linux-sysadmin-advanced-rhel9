# Advanced Linux Sysadmin - RHEL 9

Course materials and exercise resources for the Advanced Linux System Administration course (RHEL 9, 2026 Edition).

## Quick Start

```bash
git clone https://github.com/jnuyens/linux-sysadmin-advanced-rhel9.git
cd linux-sysadmin-advanced-rhel9
```

## Structure

- `chapter1/` - systemd, startup & shutdown
- `chapter2/` - System administration tools & remote management
- `chapter3/` - Package management, repositories & lifecycle

Each chapter directory contains template files, scripts, and configuration examples referenced in the exercises.

## Requirements

- RHEL 9 (or compatible: AlmaLinux 9, Rocky Linux 9)
- Root or sudo access
- Network access for package installation
